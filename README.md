Здесь лежат исходники для отладки паресера.

Иерархия папок:

   >root folder
	>|
	>|-----Parser
	|	|
	|	|-----jamroot.jam
	|	|
	|	|-----src/
	|
	|
	|-----Parser_Debug
		|
		|-----jamroot.jam
		|
		|-----src/

Запуск через консоль "# b2"

В папке build/bin будет лежать файл "parser_debug" или "parser_debug.exe"(под windows), его и надо запускать.
